事件：就是文档或浏览器窗口中发生的一些特定的交互瞬间
比如：点击某个元素、鼠标悬浮在某个元素之上、关闭弹窗、拖拽某个元素等等

Javascript 是以事件驱动为核心的一门语言。它与 HTML 之间的交互通过事件实现

事件的三要素
事件源：引发后续事件的 html 标签（谁发生）
事件：定义的 JS 行为，比如点击、双击等（怎样发生）
事件驱动程序：对样式和 HTML 的操作，DOM（发生啥）
比如：网页上弹出广告，点击右上角的 x，广告关闭了。事件源是 x，事件是 onclick，事件驱动程序是：广告关闭

事件代码书写
1. 获取事件源 document 上获取元素
2. 绑定事件 事件源.事件 = function () {事件驱动程序} 通常就是函数，可以是匿名的也可以是已定义的
3. 编写事件驱动程序 关于 DOM 的一些操作

常见的事件如下：
事件名 描述
onclick 鼠标点击
ondbclick 鼠标双击
onkeyup 按下并释放键盘上的一个键时触发
onchange 文本内容或下拉菜单中的选项发生改变时
onfocus 获得焦点，表示文本框等获得鼠标光标
onblur 失去焦点，表示文本框等失去鼠标光标
onmouseover 鼠标悬停，即鼠标停留在图片等的上方
onmouseout 鼠标移除，即离开图片等所在的区域
onload 网页文档加载事件
onunload 关闭网页时
onsubmit 表单提交事件
onreset 表单重置事件
